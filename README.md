### Overview\
Energy-Flow Cosmology (EFC) is a unified thermodynamic model of the universe, describing structure, dynamics, and cognition as continuous expressions of energy flow through entropy gradients.\
\
This repository maintains structured JSON-LD schemas, reference content, and published article data used on [energyflow-cosmology.com](https://energyflow-cosmology.com).\
