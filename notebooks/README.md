# **Jupyter Notebooks — Energy-Flow Cosmology**

Interactive notebooks for exploring and validating the **Energy-Flow Cosmology (EFC)** framework.
They provide baseline simulations, comparisons against ΛCDM, and tests of entropy-driven dynamics.

---

## **Contents**

| Notebook                                | Purpose                                                                                                                                |
| --------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| **EFC_Baseline.ipynb**                  | Simulates the EFC expansion rate (H(S)) and the energy-flow potential (E_f) as functions of entropy. Establishes a numerical baseline. |
| **EFC_vs_LCDM.ipynb**                   | Plots EFC and ΛCDM expansion curves together to show where thermodynamic evolution deviates from the standard model.                   |
| **EFC_Entropy_Test.ipynb** *(upcoming)* | Tests extreme entropy limits: (S \rightarrow 0) (high-order regime) and (S \rightarrow 1) (equilibrium). Useful for boundary analysis. |

---

## **Usage**

You can run these notebooks locally or inside VS Code / JupyterLab / GitHub Codespaces.

### Install dependencies:

```bash
pip install numpy matplotlib
```

### Start Jupyter:

```bash
jupyter lab
```
