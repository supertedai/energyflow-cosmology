# EFC–H Structural Model

This directory contains the master and minimal formal specifications for
the EFC–H structural model. This layer defines structural formation,
halo profiles, resistance coupling, and propagation behaviour in
Energy-Flow Cosmology.

## Files

- `index.tex` — Master formal specification  
- `minimal_spec.tex` — Minimal reference  
- `index.jsonld` — Semantic metadata  
- `schema.json` — Validation schema
