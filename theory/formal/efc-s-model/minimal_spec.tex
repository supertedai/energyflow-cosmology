\section*{Minimal S Model}

Structural equilibrium:

\[
\nabla E_f = 0
\]

With:

\[
E_f = \rho(1 - S)
\]

Thus:

\[
\rho(r) = \frac{E_0}{1 - S(r)}
\]

Resistance:

\[
R = R(S)
\]

Propagation:

\[
c_{\text{eff}} = (1 + R)^{-1}
\]
