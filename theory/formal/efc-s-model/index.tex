\section*{EFC–S: Structural Layer}

The structural model defines the fixed background fields that Energy–Flow
Cosmology builds on. These fields are not dynamic solutions; they specify the
baseline state into which energy–flow dynamics (EFC–D) operate.

% ------------------------------------------------------------
\subsection*{Grid State}

The grid state is written as:
\[
G(x)
\]
representing the local structural capacity of spacetime.

% ------------------------------------------------------------
\subsection*{Entropy Field}

The entropy profile is:
\[
S(x),
\]
a scalar field describing local disorder and information capacity.

% ------------------------------------------------------------
\subsection*{Density Field}

Mass density is:
\[
\rho(x) = \frac{m(x)}{V(x)}.
\]

% ------------------------------------------------------------
\subsection*{Resistance Profile}

The structural resistance is:
\[
R(x) = f\!\left(G(x), S(x)\right),
\]
governing how energy and light move through the grid.

% ------------------------------------------------------------
\subsection*{Baselines}

The core structural relation is:
\[
\Phi_S(x) = \rho(x)\,\bigl(1 - S(x)\bigr),
\]
which defines the structural potential used in EFC–D.

This specification defines only the structural layer.  
Dynamical relations appear in the EFC–D model.
