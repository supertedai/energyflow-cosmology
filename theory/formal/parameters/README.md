# EFC Global Parameters

This directory defines the global parameters, constants, and coefficients
used across the formal models of Energy-Flow Cosmology. These include
entropy normalization, coupling constants, structural constants,
simulation parameters, and stability thresholds.

## Files

- `index.tex` — Master parameter specification  
- `minimal_spec.tex` — Minimal reference version  
- `index.jsonld` — Semantic metadata  
- `schema.json` — Validation schema  

Version: 1.0
