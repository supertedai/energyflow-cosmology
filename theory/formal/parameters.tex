\section{Parameter Definitions}

This section lists the parameters used in the EFC model.

\begin{itemize}
    \item $S_{\min}, S_{\max}$ : bounds of the entropy coordinate.
    \item $E_{f,0}$ : reference energy-flow normalization.
    \item $A_\Phi$ : scaling constant for the energy-flow potential.
    \item $A_H$ : normalization factor for the expansion mapping.
    \item $k_S$ : gradient scaling for the entropy field.
\end{itemize}

All parameter defaults are defined in the implementation under:
\texttt{src/efc/parameters.py}.
