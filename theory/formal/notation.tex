\section{Notation}

The following notation is used throughout the EFC formalism:

\begin{itemize}
    \item $S(\bm{x})$ : entropy-like field.
    \item $\bm{g}_S = \nabla S$ : entropy gradient.
    \item $E_f(\bm{x})$ : energy-flow field or density.
    \item $\Phi(E_f, S)$ : energy-flow potential.
    \item $H(E_f, S)$ : expansion mapping.
    \item $\rho(\bm{x})$ : matter-energy distribution (when needed).
    \item $v(r)$ : circular velocity for SPARC validation.
    \item $z$ : redshift.
\end{itemize}
