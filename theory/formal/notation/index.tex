\section{Notation}

The Energy-Flow Cosmology (EFC) framework uses the following central
quantities:

\begin{itemize}
    \item $S(\mathbf{x})$ : entropy field or entropy coordinate.
    \item $\nabla S$ : entropy gradient, source of structural asymmetry.
    \item $E_f(\mathbf{x})$ : energy-flow field; local density of directional energy flux.
    \item $\Phi(E_f, S)$ : energy-flow potential governing structure.
    \item $H(E_f, S)$ : expansion mapping derived from the flow-strength.
    \item $v(r)$ : circular velocity used in SPARC validation.
    \item $z$ : cosmological redshift.
    \item $\rho(\mathbf{x})$ : matter/energy distribution (only used for comparison).
\end{itemize}

Coordinates:

\begin{itemize}
    \item $\mathbf{x}$ : spatial 3-vector.
    \item $r$ : radial coordinate for rotation curve modelling.
\end{itemize}
