# Symbiotic Process (LLM)

This directory defines the Symbiotic Process for LLM interaction used in the Energy-Flow Cosmology (EFC) project.

The Symbiotic Process describes how a human and an LLM engage in a stable, structured reasoning loop, ensuring clarity, separation of roles, and reproducibility across updates.

Files:
- `index.md` — full specification
- `index.jsonld` — semantic JSON-LD node
