# EFC Core Layer

The core layer contains the fundamental computational components used across the Energy-Flow Cosmology (EFC) system.

It defines:
- the parameter set (`EFCParameters`)
- the central computational model (`EFCModel`)
- base operations for energy flow, entropy gradients, and rotation velocity
- parameter loading utilities

Contents:
- `efc_core.py` — main implementation
- `index.md` — core layer overview
- `index.jsonld` — semantic description node
