# EFC — Can Energy Flow Be Observed in Galactic Clusters?

This directory contains the paper examining whether direct or indirect
signatures of the energy–flow field (Ef) can be detected in the dynamics,
temperature gradients, entropy distribution, or lensing profiles of 
galaxy clusters.

**Included files**
- PDF (authoritative)
- Markdown (readable)
- JSON-LD metadata
- Local schema
- Machine index
- Citations file
