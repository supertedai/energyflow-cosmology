\subsection*{Overview}

Observable predictions arise directly from $S(r)$, $n(S)$, and the effective
potential $\Phi_E$.

\subsection*{Rotation Curves}

\[
v(r) = \sqrt{ r \, \partial_r \Phi_E(r) }.
\]

\subsection*{Lensing Angle}

\[
\theta = \int \nabla n(S) \, ds.
\]

\subsection*{Expansion Signal}

\[
H(S) = \sqrt{|E_f(S)|}(1+S),
\]

used when comparing with DESI, Euclid, JWST, and weak-lensing surveys.
