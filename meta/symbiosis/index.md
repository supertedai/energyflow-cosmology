# Symbiosis Metadata Index

This index file provides a human-readable overview of the Symbiosis subsystem within the EFC ecosystem.

## Purpose

- Describe the structure of the Symbiosis metadata layer.
- Provide navigation to JSON-LD and schema files.
- Serve as an entry point for integration with external platforms, APIs and documentation systems.

## Components

- **symbiosis_metadata.jsonld**  
  Primary semantic description of the Symbiosis runtime.

- **index.jsonld**  
  Top-level metadata index linking all Symbiosis components.

- **schema.json**  
  Auto-generated mapping of schema definitions.

## Related Systems

- EFC core theory (mathematical & conceptual)
- EFC API ecosystem (unified-api, query-api, graph-rag-api)
- Memory infrastructure (Neo4j, Qdrant)
