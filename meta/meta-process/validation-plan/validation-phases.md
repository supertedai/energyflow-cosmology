# Validation Phases

Validation proceeds in clearly defined phases.

---

## Phase 1 — Structural Verification
- verify all layer boundaries  
- confirm role separation  
- check transitions and rules  

---

## Phase 2 — Coherence Verification
- evaluate coherence metrics  
- detect cross-layer mismatches  
- validate boundary integrity  

---

## Phase 3 — Stability Testing
- check behaviour under high load  
- validate state persistence  
- test collapse isolation  

---

## Phase 4 — Transition Validation
- test expansion → consolidation  
- test consolidation → resonance  
- test resonance → fragmentation  
- test fragmentation → recovery  

---

## Phase 5 — Semantic Validation
- verify identifiers  
- check consistency across JSON-LD  
- ensure global schema connectivity  

---

## Phase 6 — Final Review
- summarise validation outcomes  
- flag issues for Process-of-Process  
- confirm overall correctness
